
# Ascii Art Me

This is AsciiArt auto generator.
asciiartme.py converts image to asciiart.

## Requirements

- numpy
- PIL

## Usage

```
python asciiartme.py <imagename>

python asciiartme.py miku.png  # this outputs miku.html

```
Output : asciiart html file.
